# Audio

Store narration and BGM here.

Required final files:

- `narration.mp3`: Chinese male narration generated from `scripts/narration.txt`.
- `bgm.mp3`: Locally generated low-frequency documentary music bed.

`narration.wav` is generated by the workflow for duration checks and archive use, but the HyperFrames composition uses the smaller MP3 file for faster preview and render startup.

The current BGM is generated locally with FFmpeg and does not require external music attribution.
